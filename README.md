Sales Performance Dashboard (Power BI)

This project analyzes retail sales performance using Microsoft Power BI to identify revenue trends, regional performance, and product profitability.

Tools Used

Power BI

DAX

Data Visualization


Key Features

Revenue, Profit, and Profit Margin KPIs

Monthly sales trend analysis

Regional and category performance comparison

Interactive filtering using slicers

Project Files

Sales_Performance_Dashboard.pbix – Power BI report

Dashboard export preview

Objective

To demonstrate business intelligence and data visualization skills by transforming raw sales data into actionable insights.
